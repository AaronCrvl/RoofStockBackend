# RoofStock

O objetivo deste projeto é desenvolver uma aplicação Web Api backend feita em Asp Net Core, baseada na arquitetura MVC com camadas extras para validação de regras de negócio e validação de dados que visa fornecer uma base para 
uma aplicação web (acessada via mobile ou desktop) de gerenciamento de estoques, formando assim, de maneira geral, uma arquitetura Client-server (cliente-servidor).
